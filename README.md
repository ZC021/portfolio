# 김혁진 포트폴리오

DevOps / 클라우드 엔지니어 포트폴리오입니다.

운영 업무를 하면서 만들었던 자동화, 점검, 대시보드, 접근 권한 처리 프로젝트를 모아둔 포트폴리오입니다. 회사명, 내부 주소, 계정, 운영 데이터는 제외했고 화면 이미지는 필요한 부분을 마스킹했습니다.

## 폴더별 설명

- `index.html`: 포트폴리오 본문, 프로젝트 설명, 검증 로그 요약, GitHub 링크를 담은 정적 페이지입니다.
- `img/`: 포트폴리오에 들어가는 화면 이미지입니다. 내부 계정, 문서명, 호스트명은 블러 또는 마스킹 처리했습니다.
- `.gitignore`: 로컬 편집 중 생길 수 있는 불필요한 파일을 제외합니다.

## 공개 범위

- 프로젝트별 구현 범위, 운영 흐름, 검증 내용을 넣었습니다.
- 완성한 기능과 제한 사항은 각 프로젝트 README에 분리했습니다.
- 회사 내부 데이터, 운영 환경 정보, 시크릿, 인증 파일은 포함하지 않았습니다.
